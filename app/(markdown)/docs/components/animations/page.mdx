## AnimateText

Dynamic text animation that cycles through words with visual indicators.

### Preview

<div className="my-8 p-6 border rounded-lg">
  <AnimateText words={["Dynamic", "Animated", "Text", "Component"]} />
</div>

### Usage

```tsx
import { AnimateText } from '@/components/animations'

export default function Example() {
  return (
    <AnimateText 
      words={["Hello", "World", "Welcome"]} 
      className="text-2xl font-bold"
    />
  )
}
```

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `words` | `string[]` | - | Array of words to cycle through |
| `className` | `string` | `""` | Additional CSS classes |
| `duration` | `number` | `2000` | Duration for each word (ms) |

## Animation Principles

### Performance
- Hardware accelerated transforms
- Minimal DOM manipulations
- Proper cleanup on unmount

### Accessibility
- Respects `prefers-reduced-motion`
- Keyboard navigation support
- Screen reader friendly

### Customization
- CSS custom properties
- Tailwind class support
- Theme integration

## Best Practices

### 1. Performance
```tsx
// ‚úÖ Good - hardware accelerated
<motion.div style={{ transform: 'translateX(100px)' }} />

// ‚ùå Avoid - causes layout
<motion.div style={{ left: '100px' }} />
```

### 2. Accessibility
```tsx
// ‚úÖ Good - respects user preferences
<motion.div 
  animate={{ opacity: 1 }}
  transition={{ 
    duration: 0.3,
    ease: "easeOut"
  }}
/>
```

### 3. Cleanup
```tsx
// ‚úÖ Good - proper cleanup
useEffect(() => {
  const animation = animate(/* ... */)
  return () => animation.stop()
}, [])
```
